# Crime Prediction & Safety Alert System

A launchable static web app for visualizing city crime hotspots, safety alerts, analytics, and demo risk predictions.

## Run Locally

Open a terminal in this folder and run:

```powershell
python -m http.server 5173
```

Then open:

```text
http://localhost:5173
```

## Publish Online

### Netlify

1. Go to https://app.netlify.com/drop
2. Drag this full project folder onto the page
3. Share the generated public link

### GitHub Pages

1. Create a GitHub repository
2. Upload `index.html`, `README.md`, and `package.json`
3. Open repository Settings > Pages
4. Choose branch `main` and folder `/root`
5. Share the generated GitHub Pages URL
